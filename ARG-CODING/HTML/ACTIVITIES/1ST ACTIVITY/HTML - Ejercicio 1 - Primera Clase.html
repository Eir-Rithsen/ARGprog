Concepts reviewed in class

Among the concepts reviewed in class
we can highlight the following with their respective definitions

Web: Browsers

A web browser (web browser) is software that allows access to the Web,
interpreting the information of different types of files,
and showing it to the Internet user in an appropriate way

Web: Frontend y Backend

Frontend it is the part of web development
that is dedicated to the front end of a website,
of the design of a website, from the structure
to the styles such as colors, backgrounds,
sizes, animations and effects.

Backend it process the information that
feeds the frontend of data. It is the access layer
to the data, is the technological logic that makes a web page work.

HTML

Hypertext Markup Language, tags are used
to structure and organize the content of the page.
