Crear una página para poder realizar ingresos personales.
Ingresos de Datos Personales debe ser su título.
Se deben solicitar los siguientes datos:

  • Nombre (máximo 20 caracteres). Obligatorio
  • Apellido (máximo 25 caracteres). Obligatorio
  • DNI (solo números). Obligatorio
  • Correo electrónico. Obligatorio
  • Estado Civil. Seleccionar única con las siguiente opciones:
  • Soltera/o
  • Casada/o
  • Divorciada/o
  • Viuda/o
  • Conviviente
  • Intereses. Selección múltiple con las siguientes opciones:
  • Culturales
  • Deportivos
  • Turísticos
  • Sociales
  • Otros
  • Descripción de intereses (texto de hasta 400 caracteres).
  • País de residencia. Selección única con las siguientes opciones:
  • Argentina
  • Bolivia
  • Brasil
  • Colombia
  • Costa Rica
  • Chile
  • México
  • Nicaragua
  • Panamá
  • Paraguay
  • Perú
  • Uruguay
  • Venezuela
  • Otro
