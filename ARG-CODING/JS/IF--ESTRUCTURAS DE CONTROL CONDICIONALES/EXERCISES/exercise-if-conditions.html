EJERCICIO 1

Hacer un programa que:

  • Pida un numero (utilizando prompt)
  • Si es mayor a cero mostrar "Numero positivo"
  • Si es menor a cero mostrar "Numero negativo"
  • Si es igual a cero mostrar "Es cero"
  • Si no cumple ninguna mostrar "No es un numero!"
